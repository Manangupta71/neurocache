Memory Access Pattern — Row-Major vs Column-Major
Row-Major (Sequential)
Cache Efficiency—
Accesses memory sequentially — each new element is adjacent in memory. High spatial locality → most accesses hit the same cache line.
Column-Major (Strided)
Cache Efficiency—
Accesses memory with stride N — skips entire rows between accesses. Poor spatial locality → each access likely misses the cache.
