# neurocache
**Neural Cache Sim**  Visualize how NN matrix operations hit CPU cache in real-time. Tune matrix size, cache lines, and patterns to see performance gaps.  * **Live Animation:** Watch hits/misses across 128 lines. * **Patterns:** Compare Row-Major, Col-Major, and Tiled. * **Metrics:** Track Hit Rate, Bandwidth, and GFLOPS/s efficiency.
